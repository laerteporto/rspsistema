⚠️ Por que isso é necessário?
O link enviado no WhatsApp precisa de uma URL que o cliente consiga abrir no celular. Um caminho local file:///home/... só funciona no seu computador.

Opções gratuitas para hospedar o sistema:
 • Firebase Hosting — recomendado (já usa Firebase)
 • GitHub Pages — simples e gratuito
 • Netlify — arraste e solte a pasta

Após hospedar, cole aqui a URL gerada (ex: https://rpssystem.web.app).
